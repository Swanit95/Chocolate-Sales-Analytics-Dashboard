# Chocolate Sales Analytics Dashboard

## Project Overview
This Power BI Data Visualization mini project analyzes chocolate sales performance, product contribution, customer insights, time intelligence, and sustainability impact.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Star Schema
- Bookmarks
- Page Navigation
- Slicers

## Dashboard Pages
1. Executive Overview
2. Sales Analysis
3. Product Analysis
4. Customer Insights
5. Time Intelligence
6. SDG Impact

## Key Features
- KPI Cards
- Interactive Slicers
- Bookmark Navigation
- Dynamic Visual Switching
- Time Intelligence Measures
- SDG Goal 12 Mapping
- Business Insights
